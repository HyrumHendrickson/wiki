Summary EduWiki articles are written in standard HTML using a set of CSS classes provided by styles.css. This guide covers headings, lists, tables, code blocks, blockquotes, infoboxes, and all other formatting elements.

Headings

Use heading tags to create the section structure. The Table of Contents is automatically generated from <h2> and <h3> headings. 

<h2>Main Section</h2>
<h3>Sub-section</h3>
<h4>Minor sub-section</h4>

Rules:

  • Only one <h1> per page ‚Äî the article title.
  • Use <h2> for major sections, <h3> for sub-sections.
  • Give headings descriptive text (not just "Section 1").

Paragraphs and Line Breaks

Wrap each paragraph in <p></p> tags. Avoid using <br> for spacing ‚Äî use new paragraphs instead.

Lists

Unordered list (bullets):

<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

Ordered list (numbered):

<ol>
  <li>Step one</li>
  <li>Step two</li>
</ol>

Tables

Always include a <thead> row for column headers:

<table>
  <thead>
    <tr><th>Column 1</th><th>Column 2</th></tr>
  </thead>
  <tbody>
    <tr><td>Value 1</td><td>Value 2</td></tr>
    <tr><td>Value 3</td><td>Value 4</td></tr>
  </tbody>
</table>

Result:

Column 1Column 2
Value 1Value 2
Value 3Value 4

Code Blocks

For multi-line code, wrap in <pre><code>. For inline code, use just <code>. 

<pre><code>function hello() {
    return "world";
}</code></pre>

Blockquotes

Use <blockquote> for notable quotes or callouts:

<blockquote>
  Mathematics is the language with which God has written the universe.
  ‚Äî Galileo Galilei
</blockquote>

Result:

Mathematics is the language with which God has written the universe. ‚Äî Galileo Galilei

Summary Box

Every article should have a summary box right after the title metadata:

<div class="wiki-summary" role="note">
  <strong>Summary</strong>
  Your 2‚Äì4 sentence summary here.
</div>

Info Box

The info box floats on the right side of the content. Use it for quick facts: 

<div class="infobox" aria-label="Quick facts">
  <div class="infobox-title">Topic at a Glance</div>
  <table>
    <tr><td>Field</td><td>Mathematics</td></tr>
    <tr><td>Difficulty</td><td>Intermediate</td></tr>
  </table>
</div>

Images

Store images in the media/ folder. Use the wiki-figure class for proper styling: 

<figure class="wiki-figure">
  <img src="../media/my-image.png" alt="Descriptive alt text">
  <figcaption>Caption explaining the image.</figcaption>
</figure>

Links

Internal links (to other articles):

<a href="../articles/calculus.html">Introduction to Calculus</a>

External links (always include target="_blank" rel="noopener noreferrer"):

<a href="https://example.com" target="_blank" rel="noopener noreferrer">
  External Link
</a>
