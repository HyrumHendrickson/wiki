Interactive Components Guide

Last updated: February 27, 2026 EduWiki Admin Admin Desmos LaTeX Interactive
Admin Article — This guide explains the HTML structure of interactive components. If you are writing a new article, see the WMD Format Reference for the .wmd syntax that generates these components automatically (no HTML required).
Summary EduWiki supports several interactive components that make articles more engaging for learners: live Desmos graphing calculators, beautifully rendered LaTeX mathematics via MathJax, collapsible dropdown sections for worked examples, embedded videos, and more.

LaTeX Mathematics

EduWiki uses MathJax 3 to render LaTeX. Include the MathJax script in your article's <head>: 

<script>
  window.MathJax = {
    tex: {
      inlineMath: [['$','$'], ['\\(','\\)']],
      displayMath: [['$$','$$'], ['\\[','\\]']]
    }
  };
</script>
<script src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-chtml.js" async></script>

Inline Math

Wrap inline math in single dollar signs: $E = mc^2$ renders as $E = mc^2$.

Display Math

Wrap display (block) math in double dollar signs:

$$\int_0^\infty e^{-x^2}\,dx = \frac{\sqrt{\pi}}{2}$$

Renders as:

$$\int_0^\infty e^{-x^2}\,dx = \frac{\sqrt{\pi}}{2}$$

Desmos Interactive Graphs

EduWiki supports embedded Desmos Graphing Calculator instances. Include the Desmos API script in your <head>: 

<script src="https://www.desmos.com/api/v1.9/calculator.js?apiKey=dcb31709b452b1cf9dc26972add0fda6" async></script>

Then add a container with the desmos-container class and data attributes:

<div
  class="desmos-container"
  data-label="My Interactive Graph"
  data-height="400"
  data-expressions='[
    {"id":"f", "latex":"y = x^2", "color":"#2563eb"},
    {"id":"g", "latex":"y = 2x + 1", "color":"#dc2626"}
  ]'
></div>

Data Attributes

AttributeRequiredDescription
data-labelNoCaption shown above the graph (default: "Interactive Graph")
data-heightNoHeight in pixels (default: 400)
data-expressionsYesJSON array of Desmos expression objects
data-boundsNoJSON object: {"left":-5,"right":5,"bottom":-5,"top":5}

Expression Object Fields

FieldDescription
idUnique identifier (string)
latexThe LaTeX expression to graph
colorCSS color string (optional)
sliderBoundsObject {"min":"0","max":"10","step":"0.1"} to create a slider
hiddentrue to hide the expression from the list

Dropdown / Accordion Sections

Use the wiki-dropdown component to hide content behind a click-to-reveal button. Great for worked examples, proofs, or optional details. 

<div class="wiki-dropdown">
  <button class="wiki-dropdown-trigger" type="button">
     Click to reveal the example
  </button>
  <div class="wiki-dropdown-body">
    <p>Hidden content here. Can include anything: math, tables, code...</p>
  </div>
</div>

Result:

Hidden content here. Can include anything: math, tables, code, even Desmos graphs!

You can also pre-open a dropdown by adding the open class: <div class="wiki-dropdown open">

Embedding Videos

Embed YouTube or other videos using an <iframe>:

<figure class="wiki-figure">
  <iframe
    width="100%"
    height="360"
    src="https://www.youtube.com/embed/VIDEO_ID"
    title="Video title"
    frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowfullscreen
    style="border-radius: 6px;"
  ></iframe>
  <figcaption>Caption for the video.</figcaption>
</figure>

Replace VIDEO_ID with the 11-character YouTube video ID from the URL.

The Admin Notice Box

Admin and help articles should include an admin notice at the top:

<div class="admin-notice" role="note">
  <strong>Admin Article</strong> — Brief description of what this article is.
</div>

Highlight Spans

Use the highlight utility class to mark important inline text: <span class="highlight">important term</span> renders as important term.
