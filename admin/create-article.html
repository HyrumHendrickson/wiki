How to Create a New Article

Last updated: February 27, 2026 EduWiki Admin Admin Authoring
Admin Article — This guide is for contributors adding new articles to EduWiki. No programming knowledge is required — just copy the template and fill it in.
Summary Adding a new article to EduWiki requires two steps: (1) create an HTML file in the articles/ folder, and (2) register it in config/articles.json. That's it — the navigation, search, and table of contents are all automatic.

Overview of the File Structure

EduWiki's file structure is:

wiki/
├── index.html          ← Home page (auto-populated)
├── styles.css          ← All styles
├── scripts.js          ← All JavaScript
├── config/
│   ├── site.json       ← Site name, logo, colors
│   └── articles.json   ← Article registry (YOU EDIT THIS)
├── media/              ← Images, videos, etc.
├── articles/           ← Real article HTML files (YOU ADD HERE)
└── admin/              ← Help/admin articles

Step 1 — Create the HTML File

Copy the template below and save it as articles/your-article-name.html. Use lowercase letters and hyphens for the filename (e.g., quadratic-formula.html). 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Your Article Title — EduWiki</title>
  <link rel="stylesheet" href="../styles.css">

  <!-- Include MathJax if you use LaTeX -->
  <script>
    window.MathJax = {
      tex: { inlineMath: [['$','$']], displayMath: [['$$','$$']] }
    };
  </script>
  <script src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-chtml.js" async></script>

  <!-- Include Desmos if you use interactive graphs -->
  <script src="https://www.desmos.com/api/v1.9/calculator.js?apiKey=dcb31709b452b1cf9dc26972add0fda6" async></script>
</head>
<body>

  <a class="skip-link" href="#main-content">Skip to content</a>
  <header id="wiki-header" role="banner"></header>

  <div class="wiki-layout">
    <aside id="wiki-sidebar" aria-label="Table of contents and navigation">
      <div id="toc"></div>
    </aside>

    <main id="main-content" class="wiki-content">

      <!-- Breadcrumb -->
      <nav class="wiki-breadcrumb" aria-label="Breadcrumb">
        <a href="../index.html">Home</a>
        <span class="sep" aria-hidden="true">›</span>
        <a href="../index.html#cat-CATEGORY_ID">Category Name</a>
        <span class="sep" aria-hidden="true">›</span>
        <span aria-current="page">Your Article Title</span>
      </nav>

      <!-- Title -->
      <h1 class="wiki-title">Your Article Title</h1>
      <div class="wiki-meta">
        <span> Last updated: YYYY-MM-DD</span>
        <span> Your Name</span>
        <span class="tag">Category</span>
      </div>

      <!-- Summary -->
      <div class="wiki-summary" role="note">
        <strong>Summary</strong>
        Write a 2–4 sentence summary of the article here.
      </div>

      <!-- Optional: Info Box -->
      <div class="infobox" aria-label="Quick facts">
        <div class="infobox-title">Quick Facts</div>
        <table>
          <tr><td>Key 1</td><td>Value 1</td></tr>
          <tr><td>Key 2</td><td>Value 2</td></tr>
        </table>
      </div>

      <!-- Main Content -->
      <h2>First Section</h2>
      <p>Your content here...</p>

      <h2>Second Section</h2>
      <p>More content...</p>

      <!-- Citations -->
      <section class="wiki-citations" aria-labelledby="citations-heading">
        <h2 id="citations-heading">References</h2>
        <ol>
          <li>Author, A. (Year). <em>Title</em>. Publisher.</li>
        </ol>
      </section>

    </main>
  </div>

  <footer id="wiki-footer" role="contentinfo"></footer>
  <script src="../scripts.js"></script>
</body>
</html>

Step 2 — Register in articles.json

Open config/articles.json and add a new entry to the "articles" array. Copy this template: 

{
  "id": "your-article-id",
  "title": "Your Article Title",
  "category": "mathematics",
  "path": "articles/your-article-name.html",
  "summary": "A one-sentence summary that appears in search and on the home page.",
  "tags": ["keyword1", "keyword2"],
  "author": "Your Name",
  "lastUpdated": "2026-02-27",
  "featured": false
}

Field Reference

FieldRequiredDescription
idYesUnique identifier (lowercase, no spaces)
titleYesDisplay name of the article
categoryYesMust match a category id in the categories list
pathYesPath from wiki root to the HTML file
summaryYesShort description for cards and search results
tagsRecommendedSearch keywords
authorRecommendedArticle author(s)
lastUpdatedRecommendedISO date (YYYY-MM-DD)
featuredNoSet to true to show on the home page

Adding a New Category

To add a new category, add an entry to the "categories" array in articles.json: 

{
  "id": "physics",
  "label": "Physics",
  "icon": ""
}

Then use "category": "physics" in your article entries. The new category will appear automatically in the sidebar and on the home page.

Customizing the Site

To change the site name, logo, colors, or footer text, edit config/site.json: 

{
  "name": "MyWiki",
  "tagline": "Learning Made Simple",
  "logo": "media/my-logo.svg",
  "primaryColor": "#16a34a",
  "footerText": "© 2026 MyWiki"
}

The primaryColor accepts any valid CSS color value. All buttons, links, and accents will update automatically.

Checklist for a New Article

  • Create articles/your-article.html from the template
  • Add an entry to config/articles.json
  • Write the summary box
  • Add at least 2 <h2> sections (for the Table of Contents)
  • Add citations in the .wiki-citations section
  • Preview the article in a browser
